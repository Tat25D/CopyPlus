CopyPlus - это легковесная, стильная и быстрая утилита для Windows, которая помогает в работе с шаблонными текстами. Программа вызывается быстрыми клавишами. Плавающая полупрозрачная панель с наборами слов и фраз появляется поверх любого окна (Word, браузер, мессенджеры, IDE), позволяя вставлять склонённые фразы в два клика. Формируйте слова или фразы в словари, сохраняйте их для повторного использования. Настраивайте словари и UI под себя, используйте (Alt+F1, 1-8, Space). Если программы нужна для разных задач и словараей оказалось мало - сделайте копию прграммы и добавьте в неё другие словари. Например, можно создать отдельную копию для работы с нейросетями и разместить в словарях нужные промпты для ускорерия набора текста.

✨ Ключевые особенности<br>
🎯 Умная вставка: Выделяешь текст -> нажимаешь хоткей -> выбираешь падеж -> текст мгновенно вставлен в курсор с правильным регистром и пробелом.<br>
⌨️ Полное управление с клавиатуры: Не отрывая рук от клавиатуры переключай словари (1-8), меняй регистр (Пробел) и закрывай окна (Esc).<br>
🔄 Живое переключение регистров (Space): Уникальная фича — меняет регистр прямо внутри открытого выпадающего меню без его закрытия (Без изменений ➔ со строчной ➔ С заглавной ➔ ВСЕ ЗАГЛАВНЫЕ).<br>
📚 8 независимых словарей: Разделяй юридические шаблоны, бытовые фразы и рабочие термины по разным словарям.<br>
🎨 Глубокая кастомизация UI: Темная тема "Obsidian", градиенты, ручная палитра на 48 цветов, настройка прозрачности, скругления углов, шрифтов и жирности.<br>
✂️ Интеллектуальное сокращение: Режим "Кратко" обрезает длинные фразы (например, Заявлен-ьння), разделяя логику для слов и целых фраз.<br>
🧠 Автоматическое склонение: Встроенный морфологический анализатор русского языка (на базе pymorphy3) для автоматического заполнения пустых падежей.<br>
📝 Алиасы: Используй короткие метки (<<Заявл.>>) для длинных фраз на кнопках.<br>
<br>
⌨️ Горячие клавиши<br>
Alt + F1 - (по умолчанию)	Вызвать / Скрыть парящее окно под курсором.<br>
Настроенный хоткей	- Основной хоткей (можно изменить в редакторе).<br>
1 ... 8	- Быстрое переключение между словарями 1-8.<br>
Space	- Цикличное переключение режима регистратора (Без изменений ➔ со строчной ➔ С заглавной ➔ ВСЕ ЗАГЛАВНЫЕ). Работает даже внутри открытого меню!<br>
Esc	- Закрытие: 1-е нажатие закрывает меню падежей (если открыто), 2-е нажатие закрывает само парящее окно.<br>

🛠️ Установка и запуск<br>
Убедитесь, что у вас установлен Python 3.10 или выше.<br>
Клонируйте репозиторий:
<code>
git clone https://github.com/Tat25D/CopyPlus.git <br>
</code>
Установите зависимости (рекомендуется использовать виртуальное окружение): 
<code>
pip install -r requirements.txt <br>
</code>
Запустите программу:
<code>
python main.py <br>
</code>
(При первом запуске в папке data/ автоматически создадутся файлы phrases.json и settings.json)<br>
